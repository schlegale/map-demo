# Geolocation Monorepo Demo 🌍

A full-stack, location-based application utilizing a monorepo architecture. This project serves as a technical demonstration of full-stack integration.

## 🚀 Tech Stack

- **Mobile:** React Native (Expo), React-Native-WebView, Expo-Location
- **Web:** React.js
- **Backend:** Python, Flask
- **Database:** PostgreSQL

## 📂 Project Structure

This project uses a monorepo setup to keep the frontend, mobile, and backend codebases organized in a single repository.

```text
map-demo/
├── apps/
│   ├── mobile/       <-- React Native application (iOS/Android)
│   ├── web/          <-- React web application
├── package.json
├── .gitignore
└── README.md
```
